# Parasik Shop (Django SSR + jQuery)

An online shop platform for mobile phones and laptops built using **Django (Server‑Side Rendering)** and **jQuery**.  
The project focuses on simplicity, clean structure, high performance, and easy customization.

---

## 🚀 Features

- Full Server-Side Rendered (SSR) pages using Django Templates  
- jQuery-powered dynamic UI interactions  
- Product listing for mobiles and laptops  
- Product details page  
- Add to cart (AJAX)  
- Responsive and modern UI  
- Fully extendable backend structure  


---

## 🛠️ Technologies Used

- **Python 3.x**
- **Django**
- **Django Template Engine (SSR)**
- **jQuery**
- **HTML5 / CSS3**
- **tailwind css**
